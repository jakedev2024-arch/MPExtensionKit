# MPExtensionKit

MPExtensionKit provides extension utilities for iOS development.

## Requirements

- iOS 14.0+
- Swift 5.7+

## Installation

### Swift Package Manager

Add the following to your `Package.swift` file:

```swift
dependencies: [
    .package(url: "https://github.com/yourusername/MPExtensionKit.git", from: "1.0.0")
]
```

Or add it through Xcode:
